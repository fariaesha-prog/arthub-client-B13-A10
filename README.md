## 📌 Project Description

**ArtHub** is a digital platform that connects art lovers, collectors, and buyers with talented artists. The platform allows users to browse, discover, and purchase original artworks. Artists can upload, manage, and showcase their creations, while admins oversee the entire system to ensure smooth and secure operations.

---

## 💡 Purpose of the Project

Traditional art buying is often limited to physical galleries or exhibitions. ArtHub solves this by creating an online marketplace that:

- Democratizes access to art globally  
- Helps emerging artists reach a wider audience  
- Provides a secure and seamless purchasing experience  
- Demonstrates modern full-stack development using the MERN stack  

This project highlights advanced concepts such as role-based authentication, secure backend APIs, and interactive UI features.

## ✨ Key Features

### 👤 Users
- Browse and discover artworks
- View detailed artwork pages
- Purchase artworks securely
- Comment on artworks
- Save favorite artworks

### 🎨 Artists
- Upload new artworks
- Edit and delete listings
- Manage personal gallery
- Track engagement (views, likes, purchases)

### 🛡️ Admin
- Manage users and artists
- Moderate artworks and content
- Monitor platform activity
- View system analytics

### ⚙️ System
- Role-based authentication (User / Artist / Admin)
- Secure login & JWT authentication
- Responsive UI for all devices
- REST API integration
- Payment-ready structure

## 🚀 Getting Started

### 1. Clone the repository
```bash
git clone https://github.com/fariaesha-prog/arthub-client-B13-A10.git
```

### 2. Install dependencies

#### Frontend
```bash
cd client
npm install
```

#### Backend
```bash
cd server
npm install
```

### 3. Setup environment variables

Create a `.env` file inside the backend folder:

```
PORT=5000
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
```

### 4. Run the project

#### Start backend
```bash
npm run dev
```

#### Start frontend
```bash
npm run dev
```
